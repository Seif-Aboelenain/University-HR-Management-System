# University HR Management System

A full-stack web application developed using **ASP.NET (C#)** and **SQL Server** to manage university HR operations.

## Features
- Employee management system
- Admin, HR, and Academic portals
- Role-based access
- Payroll and database operations
- 30+ stored procedures

## Tech Stack
- ASP.NET Web Forms (C#)
- SQL Server
- HTML / CSS / JavaScript

## How to Run

1. Open the project in **Visual Studio**
2. Open SQL Server Management Studio (SSMS)
3. Run:
   - `Final (1).sql`
   - `Insertions -MS3.sql`
4. Make sure database name matches `Web.config`
5. Run the project

## Project Structure
- `Academic / Academic2` → Academic modules
- `Admin` → Admin portal
- `HR` → HR module
- `Scripts / Content` → Frontend assets
- `.sql files` → Database setup
